# Dream City Finder - https://dream-city-finder.herokuapp.com/ <br/>
Datathon 2020 - Dream City finder <br/>
This app lets you filter US cities based on rent range and gives statistics about the diversity of that city to help the user find the perfect city according to their perfernces! <br/>
The app can be found at https://dream-city-finder.herokuapp.com/ <br/>
To run the code, clone this repository and run the main.py file from the command line using the command: <br/>
streamlit run main.py
